# 🔐 Password Generator

A simple command-line password generator built with Python.  
You can customize the length and choose whether to include numbers and symbols.

---

## 🚀 Features

- Set password length (8–64 characters)
- Toggle numbers and symbols on/off
- Guarantees at least one character from each selected pool
- Generate multiple passwords in one session

---

## 🛠 Built With

- Python 3
- `random` module (standard library)
- `string` module (standard library)

> No external libraries needed!

---

## ▶️ How to Run

1. Make sure Python is installed on your system
2. Clone this repo or download the file
3. Open a terminal in the project folder and run:

```bash
python password_generator.py
```

---

## 💻 Example

```
=== Password Generator ===
Password length (8–64): 16
Include numbers?  (y/n): y
Include symbols?  (y/n): y

  ✔  Your password: k$3Lp!wQ9#mZrT2@

Generate another? (y/n): n
Bye!
```
